Para executar o projeto siga os seguintes passos:

- Clone o repositório.
- Crie um ambiente virtual com `python -m venv venv`
- Instale Flask e TinyDb
- Rode o projeto com `cd src` e logo após `python -m flask --app routes run`

O projeto utiliza as bibliotecas Flask e TinyDb
